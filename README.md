* **Step 1:** \
  To download this script, it is necessary to have git installed. If you don't
  have git already installed, or if you are unsure, run the following command:
  이 스크립트를 다운로드하려면 git이 설치되어 있어야 합니다. git이 아직 설치되지 않았거나 확실하지 않은 경우 다음 명령을 실행하세요:

```shell
sudo apt-get update && sudo apt-get install git -y
```

* **Step 2:** \
  Once git is installed, use the following command to download KIAUH into your
  home-directory:
  git이 설치되면 다음 명령을 사용하여 KIAUH를 홈 디렉토리에 다운로드합니다:

```shell
cd ~ && git clone https://github.com/MOMENT3D/kiauh.git
```

* **Step 3:** \
  Finally, start KIAUH by running the next command:
  마지막으로 다음 명령을 실행하여 KIAUH를 시작합니다:

```shell
./kiauh/kiauh.sh
```
